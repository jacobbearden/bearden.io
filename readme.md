# [bearden.io](http://bearden.io)

> My little corner of the internet.


## about

- content in [markdown](//daringfireball.net/projects/markdown), built with [jekyll](//jekyllrb.com).

## install locally

```bash
git clone git@github.com:jacobbearden/bearden.io.git
cd bearden.io
bundle install
jekyll serve
```
